# PrivacyTips
Software recommendations for privacy in general 

## Android

### Browsers
#### Brave Browser
Brave browser is open source, blocks fingerprinting and trackers as much as possible and is improving in each update.

#### Bromite
Bromite is chromium based browser that removed all bad privacy things from chromium. It's blazing fasts, blocks ads, trackers and fingerprinting too. It is also open source and you can download from F-droid repo or their <bold><a href="https://www.bromite.org"> website </a></bold>
